📘 UniPilot – University Management System
📌 Description

UniPilot is a web-based university management system developed with Laravel. It allows administrators, professors, and students to efficiently manage academic activities such as attendance, courses, schedules, and academic records.

🚀 Features
👨‍🎓 Student Management
Add, edit, delete students
View student profiles
Track academic records
👨‍🏫 Professor Management
Manage professor accounts
Assign courses
Track teaching activities
📊 Attendance System
Mark student attendance
View attendance history
Generate attendance reports
📚 Academic Tools
Pedagogical journal management
Course organization
Administrative settings
🔐 Authentication
Secure login system
Role-based access (Admin / Professor / Student)
🛠️ Tech Stack
Backend: Laravel
Frontend: Blade (Laravel templates)
Database: MySQL
Languages: PHP, JavaScript, HTML, CSS
Tools: Composer, Node.js, npm
